## 📌 Overview

This repository contains the code and results for my bachelor thesis:

**"Shapelet-Based Analysis of Human Activities (Repetitive vs Non-Repetitive)"**

The project investigates how shapelet-based time-series classification methods behave on different types of human activities, comparing periodic locomotion activities and non-periodic gesture-based activities.
